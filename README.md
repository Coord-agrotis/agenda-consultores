# Agenda de Consultores

Controle semanal de alocacao de consultores em clientes, indicando se o atendimento e presencial ou remoto. Um consultor pode ser alocado em mais de um cliente na mesma semana.

## Como usar

1. Abra o arquivo `index.html` no navegador (duplo clique ou "Abrir com" o navegador de sua preferencia).
2. Cadastre os consultores e os clientes nas secoes correspondentes.
3. Navegue entre as semanas usando as setas no topo da pagina.
4. Para cada consultor, escolha um cliente e a modalidade (presencial/remoto) e clique em "Alocar". Repita para alocar o mesmo consultor em mais de um cliente na semana.
5. Os dados ficam salvos no navegador (localStorage) e sao mantidos entre as visitas, na mesma maquina e navegador.

## Limitacoes

- Os dados sao armazenados localmente no navegador, sem sincronizacao entre maquinas ou usuarios.
- Nao ha autenticacao ou controle de acesso.
