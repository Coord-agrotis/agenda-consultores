    Sincronizacao com Google Sheets

    Conecte a uma planilha do Google Sheets para que voce e seus colegas usem os mesmos dados, em qualquer computador. Isso exige criar uma credencial gratuita no Google Cloud (uma unica vez):

    1. Acesse console.cloud.google.com, crie um projeto (ou use um existente).
    2. Em "APIs e servicos > Biblioteca", ative a Google Sheets API.
    3. Em "APIs e servicos > Credenciais > Criar credenciais > ID do cliente OAuth", escolha tipo Aplicativo da Web e adicione a origem de onde voce abre este arquivo em "Origens JavaScript autorizadas" (ex: o link do Google Drive/hospedagem, ou null se for um arquivo local).
    4. Cole o Client ID gerado abaixo e clique em Conectar. Sera pedida sua autorizacao do Google na primeira vez.
